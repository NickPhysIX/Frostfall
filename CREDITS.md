# Frostfall — Credits and Asset Notes

Frostfall is a small hobby/browser-game project by Niels Braakman, built iteratively with AI-assisted coding, design, QA, audio and asset support.

## Project direction

Niels was in the director's seat throughout the project. He chose the feature direction, requested changes, selected what to keep or discard, coordinated feedback between multiple AI tools, downloaded and supplied external sound effects, tested playable builds, and made the final calls on balancing, style and packaging.

The practical build work was AI-assisted. ChatGPT, Claude, Gemini and Grok were used in different roles across the project: implementation, code suggestions, QA, balance review, audio/music ideation, sprite/pixel-art generation support, packaging, changelog and release-note maintenance. Some code integrated into the package originated from suggestions or snippets produced by Gemini, Claude or Grok; a large part was implemented directly by ChatGPT during the iterative build sessions.

## Working roles

- **Niels Braakman**: project direction, concept ownership, feature requests, playtesting, acceptance decisions, asset selection, SFX sourcing, final judgement and release packaging decisions.
- **ChatGPT**: primary package builder in this session; implemented many code changes, assembled zip packages, integrated assets, updated documentation, changelogs and release notes.
- **Claude**: QA/review partner and source of several patch proposals, architecture notes, bug diagnoses and balancing suggestions.
- **Gemini**: source of music-loop generation, visual/style ideas, SFX/audio workflow suggestions, code snippets and pixel-art/asset generation support.
- **Grok**: source of additional balance/code review and feedback on selected versions.
- **Apple Photos**: used by Niels for manual color variations and sprite/color refinements.

All AI proposals were requested, filtered and directed by Niels.

## Audio

- **Sound effects**: sourced by Niels from Pixabay.
- **Music loops**: generated by Niels with Gemini.
- Audio integration, playback logic, mute/background behavior, cue timing and wave-audio logic were developed iteratively with AI assistance.

## Visual assets

- **Pixel-art sprites and game visuals**: generated and refined with help from Gemini and ChatGPT.
- **Color variations**: adjusted by Niels in Apple Photos.
- **App icons and victory/end-screen imagery**: developed through the same AI-assisted pixel-art workflow and manual selection/refinement.

## Development

- Core game code, balancing, UI/UX, PWA packaging, changelog/release notes and iterative patches were developed collaboratively by Niels using ChatGPT, Claude, Gemini and Grok.
- Frostfall is a prototype/hobby project and not intended as a commercial asset pack.
