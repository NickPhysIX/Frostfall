# Frostfall — Credits and Asset Notes

Frostfall is a small hobby/browser-game project by Niels Braakman, built iteratively with AI-assisted coding, design and asset support.

## Audio

- **Sound effects**: sourced by Niels from Pixabay.
- **Music loops**: generated by Niels with Gemini.
- Audio integration, playback logic, mute/background behavior and cue timing were developed iteratively with AI assistance.

## Visual assets

- **Pixel-art sprites and game visuals**: generated and refined with help from Gemini and ChatGPT.
- **Color variations**: adjusted by Niels in Apple Photos.
- **App icons and victory/end-screen imagery**: developed through the same AI-assisted pixel-art workflow and manual selection/refinement.

## Development

- Core game code, balancing, UI/UX, PWA packaging, changelog/release notes and iterative patches were developed collaboratively by Niels using ChatGPT, Claude and Gemini.
- Frostfall is a prototype/hobby project and not intended as a commercial asset pack.

## Note

For any public or commercial reuse, check the license/usage terms for the original sound-effect downloads and any generated assets used in the project.
