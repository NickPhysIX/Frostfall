# Frostfall v9.9.3 — Tower Caps Balance

## Summary
This update adds tower caps to keep the late game and New Game+ from becoming too passive or overpowered.

Walls/palisades remain unlimited, but DPS/control towers now have type-specific caps.

## Changes
- Regular towers are capped at 25 in the base game.
- Ice towers are capped at 25 in the base game.
- In New Game+, both tower caps increase to 30.
- Palisades/walls remain unlimited.
- Existing saves are grandfathered: existing towers are not removed, even if already above the cap.
- If a save is already above a cap, the player simply cannot build more towers of that type.
- Build buttons now show current count/cap.
- Build preview turns red when the selected tower type is capped.
- Floating warning appears when trying to build beyond the cap.
- Service-worker cache bumped to `frostfall-v9-9-3`.

## Save / compatibility
- No save reset required.
- Existing saves remain compatible.
