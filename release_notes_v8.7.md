# Frostfall v8.7 — Hunters & Compass

This release adds a subtle navigation layer and a new permanent helper type.

## Highlights

- White camp arrow when home is off-screen.
- Red enemy arrows for the nearest off-screen threats.
- New Hunter/Jager helper upgrade.
- Hunters patrol, shoot weak ranged projectiles, and attack in melee when enemies get close.
- Hunters are permanent and scale only with axe power.
