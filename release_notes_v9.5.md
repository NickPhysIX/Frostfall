# Frostfall v9.5 — Palisade Snow Polish

Small visual polish pass for the procedural palisade wall renderer.

## Changed

- Horizontal palisade beams now stop at the center of end/corner posts.
- Continuous snow strips now use the same center-stop behavior.
- This removes little overhanging snow/wood fragments around corners and T-junctions.
- No new sprites; the wall and snow are still generated procedurally at runtime.
- Save key unchanged from v9.4; service-worker cache bumped to `frostfall-v9-5`.
