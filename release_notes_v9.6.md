# Frostfall v9.6 — Arrow Towers & Full Compass

## Summary
This update improves combat readability and navigation clarity in the late game. Regular towers now fire arrows instead of orange orb shots, bringing them visually in line with the hunter unit fantasy. The edge-of-screen compass system has also been expanded so it can represent all off-screen enemies instead of only a small capped subset.

## Included changes
- Regular towers use arrow projectiles.
- Ice towers keep their blue slow projectile.
- Enemy compass arrows are no longer capped to a small number.
- Enemy arrow markers now scale and fade with distance.
- Boss/alpha markers stay slightly larger and more prominent.
- Service worker cache bumped to `frostfall-v9-6`.

## Save / compatibility
- No reset required.
- Existing saves should continue to work.
- SAVE_KEY unchanged from v9.4/v9.5 lineage.
