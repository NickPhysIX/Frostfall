# Frostfall v8.9 — Hunter Combat Polish

Deze update maakt de hunters visueel duidelijker in combat.

## Highlights
- Hunters tonen nu een korte spinning-axe animatie met vier bijlen bij melee-aanvallen.
- Hunters schieten op afstand nu zichtbare arrow/spear projectiles.
- De bestaande v8.8 palisade wall rendering blijft behouden.

## Installatie
Upload de inhoud van deze map naar de GitHub Pages repository. Door de service-worker cache bump naar `frostfall-v8-9` zouden browsers de nieuwe build automatisch moeten ophalen.
